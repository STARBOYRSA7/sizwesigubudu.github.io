# Sizwe Sigubudu — Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript. Designed for GitHub Pages deployment.

🌐 **Live site:** [starboyrsa7.github.io/sizwesigubudu.github.io](https://starboyrsa7.github.io/sizwesigubudu.github.io/)

---

## Overview

A dark-mode developer portfolio with a fixed sidebar layout on desktop and a mobile-responsive design with a bottom navigation bar. Built from scratch with no frameworks or dependencies — just plain HTML, CSS, and JavaScript.

---

## Features

- Fixed sidebar with contact info and navigation (desktop)
- Sticky top header + bottom tab bar (mobile)
- Active nav highlight that updates as you scroll
- Project filter — filter by Mobile, Web, or Full-Stack
- Scroll reveal animations on cards
- CV download button
- Fully responsive down to 480px

## Running Locally

No build tools needed. Just open `index.html` in your browser:

```bash
git clone https://github.com/STARBOYRSA7/sizwesigubudu.github.io.git
cd sizwesigubudu.github.io
# Open index.html in your browser
```

Or use the VS Code Live Server extension for live reload.

---

## Deploying to GitHub Pages

1. Push all files to the `main` branch
2. Go to **Settings** → **Pages**
3. Set source to `main` branch, `/ (root)`
4. Save — live in ~60 seconds
